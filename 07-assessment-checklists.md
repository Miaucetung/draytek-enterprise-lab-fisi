# Assessment-Checklists

## Übersicht

Diese Checklisten helfen bei der **Selbstbewertung** nach jedem Lab sowie bei der **Vorbereitung auf IHK-Prüfungen**. 

## Lab-Abschluss-Checkliste

### Allgemeine Checkliste (für jedes Lab)

```markdown
## Lab [Nummer]:  [Titel]

**Datum:** ________  
**Bearbeiter:** ________  
**Dauer:** ________ Stunden

### Durchführung
- [ ] Alle Schritte aus "Durchführung" absolviert
- [ ] Konfiguration dokumentiert (Screenshots, Notizen)
- [ ] Keine Schritte übersprungen

### Validierung
- [ ] Alle Validierungstests durchgeführt
- [ ] Alle Tests erfolgreich (100%)
- [ ] Testergebnisse dokumentiert (Screenshots, Command-Outputs)

### Troubleshooting
- [ ] Mindestens 3 Fehlerszenarien selbst durchgespielt
- [ ] Fehlerbilder erkannt und gelöst
- [ ] Lösungswege dokumentiert

### Konfiguration
- [ ] Config-Backup erstellt (sanitized!)
- [ ] Backup gespeichert in /config/exports/
- [ ] Credentials entfernt aus Backup
- [ ] YAML-Vorlage ausgefüllt (falls vorhanden)

### Dokumentation
- [ ] Lab-Ziele erreicht
- [ ] Alle geforderten Artefakte erstellt
- [ ] Reflexion geschrieben (Was gelernt?  Was war schwierig?)
- [ ] Dokumentation vollständig für IHK-Portfolio

### Aufräumen
- [ ] Test-Konfigurationen entfernt (falls temporär)
- [ ] System in definiertem Zustand für nächstes Lab
- [ ] Keine "hängenden" Konfigurationen

**Gesamtbewertung:** ☐ Bestanden  ☐ Nacharbeit erforderlich
